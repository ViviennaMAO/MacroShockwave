# 产品需求文档 (PRD)：Luffa MacroShockwave 小程序

## 1. 文档概述

- **产品名称**：MacroShockwave (Luffa Mini-Program)
    
- **版本**：V1.0.0
    
- **状态**：初稿
    
- **目标**：打造一个卡片式、游戏化的比特币价格预测市场小程序
    

---

## 2. 产品定位

重要金融与经济数据对比特币价格的影响预测平台

---

## 3. 用户功能需求

### 核心价值主张

- **实时事件下注**：抓住 CPI、NFP、GDP 等重大宏观事件发布时刻

- **三重博弈玩法**：数据狙击手 + 波动猎人 + 精准点位，满足不同风险偏好

### Shockwave 冲击波下注面板

实时展示即将发布的宏观经济事件，支持三种下注玩法：


| 玩法 | 名称 | 逻辑 | 特点 |

|------|------|------|------|

| A | **数据狙击手 (Data Sniper)** | 预测公布值与预期值的关系 | 鸽派/中性/鹰派，Pari-mutuel 动态赔率 |

| B | **波动猎人 (Volatility Hunter)** | 只赌波动幅度，不赌方向 | 风平浪静 vs 惊涛骇浪 |

| C | **精准点位 (Jackpot)** | 预测结算时刻的精确价格区间 | 高赔率彩票玩法，100x 潜力 |

  

**支持事件**：

- CPI (消费者物价指数)

- NFP (非农就业数据)

- GDP (国内生产总值)

- Fed Rate Decision (美联储利率决议)


**事件时间轴**：

- 下注窗口 → 锁定倒计时 → 数据发布 → 结算

### 投资组合页面 (Portfolio)

用户个人持仓管理页面：

概览卡片

- **投资组合价值** - 总资产净值

- **现金余额** - 可用余额

- **盈亏统计** - 支持 1D/1W/1M/ALL 时间筛选

  
持仓管理

- **Positions** - 当前持仓

- **Open Orders** - 未结订单

- **History** - 历史记录

持仓详情表

| 字段 | 说明 |

|------|------|

| MARKET | 市场/资产 |

| AVG → NOW | 均价 → 现价 |

| BET | 下注金额 |

| TO WIN | 潜在收益 |

| VALUE | 当前价值 |

  

### 用户系统（Profile）

  

### 认证方式

- Luffa钱包登录，无需注册

- 用户头像与用户名展示

- 登出功能

  

### 用户界面

- 顶部导航栏显示登录状态

- 已登录：显示头像 + @username + 登出按钮

## 4. 技术需求 (Luffa SDK 集成)

| **功能模块** | **技术实现要点**                                         |
| -------- | -------------------------------------------------- |
| **容器适配** | 使用 H5 响应式布局，适配 Luffa `superbox-mini-sdk` 环境。       |
| **钱包连接** | 启动小程序时调用 `luffa.getAccount()` 获取当前登录地址。            |
| **链上交互** | 调用 `luffa.sendTransaction()` 支持在小程序内直接参与下注与资产管理。   |
| **分享功能** | 集成 `luffa.setShareInfo()`，支持用户将特定数据看板分享至 Luffa 群聊。 |

- 卡片 hover 效果
